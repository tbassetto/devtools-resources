Ressources from the workshop. This page could be useful even if you weren't there. Note that only the resources that helped me are listed.

Download the bleeding edge versions

I recommend you to use nightlies version even if they can prove themselves unstable from time to time. WebKit.app (Mac) has really different developer tools than Chrome or Opera or even Safari 6.0 ;) Lastly, Internet Explorer 11 features revamped developer tools.

  • Chrome (Canary)

  • Safari (WebKit.app)

  • Firefox (Nightly)

  • Internet Explorer (11)
